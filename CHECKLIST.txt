📌 ANTES DE EMPEZAR (Solo primera vez)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

☐ Instalar Node.js 18+
    Comando: node -v
    
☐ Instalar Python 3.11+
    Comando: python --version
    
☐ Instalar XAMPP (para MySQL)
    URL: https://www.apachefriends.org/
    
☐ Instalar Chocolatey y mkcert
    En PowerShell (Admin):
choco install mkcert -y
mkcert -install

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


📌 CONFIGURACIÓN INICIAL (Solo primera vez)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. OBTENER IP LAN
☐ Ejecutar: ipconfig
☐ Anotar IPv4: ___.___.___.___
Tu IP actual: 192.168.1.72

2. GENERAR CERTIFICADOS SSL
☐ cd c:\Users\reric\OneDrive\Documentos\chat-main
☐ mkdir certs
☐ mkcert -key-file certs\local-key.pem -cert-file certs\local.pem 192.168.1.72 localhost
☐ Verificar: dir certs
Debes ver: local.pem y local-key.pem

3. CONFIGURAR BASE DE DATOS
☐ Abrir Panel XAMPP → Start MySQL
☐ Ir a: http://localhost/phpmyadmin
☐ Ejecutar archivo: database.sql
☐ Verificar tablas: users, chats, chat_members, messages

4. CONFIGURAR BACKEND
☐ cd backend
☐ python -m venv .venv
☐ .\.venv\Scripts\activate
☐ pip install -r requirements.txt
☐ copy .env.example .env
☐ notepad .env  →  Verificar credenciales MySQL

5. CONFIGURAR FRONTEND
☐ cd frontend
☐ npm install
☐ copy .env.example .env
☐ notepad .env  →  VITE_WS_URL=wss://192.168.1.72:8765

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

       New-NetFirewallRule -DisplayName "Chat Frontend" -Direction Inbound -LocalPort 5173 -Protocol TCP -Action Allow

📌 SOLUCIÓN RÁPIDA DE PROBLEMAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Backend no arranca
   → Verificar MySQL corriendo en XAMPP
   → Activar entorno: .\.venv\Scripts\activate
   → Revisar credenciales en backend\.env

❌ Frontend error WS
   → Verificar backend corriendo
   → Revisar IP en frontend\.env
   → Reiniciar frontend: Ctrl+C y npm run dev

❌ No conecta desde móvil
   → Usar IP LAN (no localhost)
   → Abrir puertos en firewall (ver arriba)
   → PC y móvil en misma red WiFi

❌ Cámara no funciona
   → Usar HTTPS (candado en navegador)
   → Permitir permisos en navegador
   → Cerrar Zoom, Teams, etc.
